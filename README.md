# Expense Tracker
An application for reviewing expenses.
You can upload and delete an expense.


# How to set it up
- Fork the repo
- Clone the repo link.
- cd into the repo directory.
- Open it with VS Code.
- Run it with the command (npm run dev) to go live.

# Technologies used;
- HTML
- CSS
- React + Vite 
- Vercel API (frontend)
